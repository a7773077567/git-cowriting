- `log`
  - 顯示 commit 記錄, head 位置等
  - [syntax](./notes/git_log.md)

  - 將檔案新增至暫存區(index)

  - 將檔案移至儲存庫(repository)

  - 切換分支, 復原檔案
  - [syntax]()
- `rebase`
  - [syntax]()
- `merge`
  - [syntax]()
- `pull`
  - [syntax]()
- `push`
  - [syntax]()
- `fetch`
  - [syntax]()

  - 將檔案從暫存區(index)或工作目錄(working tree)移除

- `mv`
  - 移動或重新命名檔案、資料夾、分支等等

- `stash`
